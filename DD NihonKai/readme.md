# NihonKai

NihonKai is a simple educational website created as an introductory web development project. Developed as part of a school assignment, the website introduces users to Japan through an interactive homepage, image carousel, and navigation between different pages.

## Features

- Multi-page website
- Responsive navigation bar
- Interactive image slideshow
- Custom styling with CSS
- JavaScript-powered carousel controls
- Google Fonts integration
- Introduction to Japanese locations and landmarks

## Built With

- HTML5
- CSS3
- JavaScript
- Google Fonts

## Running the Project

Clone the repository:

```bash
git clone https://github.com/neilthkkr/<repository-name>.git
cd <repository-name>
```

Open `Index.html` in any modern web browser.

No additional software or installation is required.

## Purpose

This project was created as an introductory web development assignment to learn the fundamentals of front-end development. It explores core concepts including HTML page structure, CSS styling, JavaScript interactivity, navigation, and responsive webpage design.
